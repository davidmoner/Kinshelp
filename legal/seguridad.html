Seguridad y Confianza

Ultima actualizacion: 28/02/2026

KingsHelp esta pensado para acuerdos simples entre vecinos. Aqui estan las medidas y recomendaciones basicas.

1. Pagos

KingsHelp no procesa dinero. Si acordais un pago en EUR, hacedlo fuera y usa metodos seguros.

2. Prevencion de fraude de reputacion

  • Para completar un match, pedimos un minimo de conversacion real (1 mensaje por cada parte).
  • Limitamos la reputacion cuando se completan demasiados servicios entre las mismas dos cuentas en poco tiempo.
  • Para desbloquear Premium por reputacion, se requiere historial con varios vecinos distintos.

3. Consejos para quedar

  • Quedad en lugares publicos cuando sea posible.
  • No compartas datos sensibles (documentos, tarjetas, etc.).
  • Si algo suena raro, cancela y reporta.

4. Contacto

Si detectas abuso o contenido ilegal, escribe a contacto@kingshelp.app.
